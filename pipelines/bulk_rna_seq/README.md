# Bulk RNA-seq Analysis Pipeline

This folder contains the lab-standard template for bulk RNA-seq analysis.

## Planned Workflow

1. Input organization and sample sheet preparation
2. FASTQ quality control
3. Adapter trimming
4. Quantification or alignment
5. Gene-level count import
6. Differential expression analysis
7. PCA, heatmap, volcano plot, and expression visualization
8. GO, KEGG, and hallmark enrichment analysis

## Input Philosophy

This pipeline should use a sample sheet and configuration file instead of hardcoded sample names, private project paths, or dataset-specific folder structures.
