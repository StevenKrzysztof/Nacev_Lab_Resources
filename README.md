# Nacev Lab Resources

This repository provides shared computational resources, analysis standards, and reusable bioinformatics pipeline templates for the Nacev Lab.

The goal is to make lab analyses more reproducible, easier to review, and easier for new lab members and collaborators to understand. This repository will contain standardized documentation, example sample sheets, configuration templates, and workflow descriptions for common bioinformatics analyses used in the lab.

## Current Pipelines

### 1. Cut&Run / CUT&Tag Analysis

A standardized workflow for chromatin profiling experiments, including quality control, trimming, alignment, filtering, signal track generation, peak calling, and downstream comparison.

### 2. Bulk RNA-seq Analysis

A standardized workflow for bulk RNA-seq analysis, including quality control, quantification, differential expression analysis, visualization, and enrichment analysis.

## Repository Goals

- Standardize lab file naming and project organization
- Reduce hardcoded paths and sample-specific scripts
- Provide reusable templates for common analyses
- Improve reproducibility across projects
- Make analysis handoff easier between lab members
- Document expected inputs, outputs, software, and parameters

## Intended Users

This repository is intended for Nacev Lab members, bioinformaticians, wet-lab scientists, students, and collaborators who need to understand or run standard lab analysis workflows.

## Important Note

This repository should not contain raw sequencing data, private patient/sample information, or hardcoded project-specific paths.
